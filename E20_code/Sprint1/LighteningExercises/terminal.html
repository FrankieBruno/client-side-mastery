White Belt

Create a directory for Martin's Aquarium name "Martins"

Yellow Belt

Navigate into your Martins directory and create the files index.html, main.css and main.js

Green Belt

Create directories inside Martins for styles and scripts. Move the appropriate files into those directories. Open the directory in VS code and add the boiler plate html to your index.html

Blue Belt

Create a directory name practice with two files in it, delete1.html and delete2.html. First delete only delete1.html then delete the whole directory along with delete2.html in 1 command.

Red Belt

Create the directory structure for a project (WHICH ONE?). Copy the boiler plate code from the Martins project you set-up in the Green Belt exercise. Read the contents of your html in the terminal.
