Task 1:Creating a file structure

Topic: Terminal

Getting comfortable with using the terminal takes both time and practice. This task is designed to give you the first bit of practice with basic commands.

Acceptance Criteria:

  • Make a directory named "pets".
  • Make a file inside the "pets" directory named "rex.html".
  • Make two directories inside the "pets" directory named "styles" and "scripts"
  • Make a file inside the "styles" directory named “dogStyles.css”
  • Navigate to the "pets" directory. Then navigate to the "styles" directory
  • List the contents of the "styles directory."

Task 4:Create a File Structure for Martin's Aquarium

Topic: Terminal

Before we start building Martin's Aquarium, we need to create the basic file structure. Use your terminal to create the file structure.

Acceptance Criteria:

  • Martin's Aquarium needs to have it's own root directory.
  • There should be an index.html file in the root directory
  • There should be directories named scripts and styles.
  • Inside the scripts directory, there should be a file named main.js
  • Inside the styles directory, there should be a file named main.css
